# 🎓 AI Powered Campus Event Management System

A full-stack web application designed to simplify campus event management by providing a centralized platform for students, organizers, and administrators.

---

## 📌 Project Overview

Traditional campus event management often relies on posters, spreadsheets, and messaging groups, making it difficult to manage registrations and communicate effectively.

This project provides a digital platform where students can browse and register for events while administrators and organizers efficiently manage event activities.

The system also includes an AI-powered recommendation engine that suggests relevant events based on student interests.

---

## ✨ Features

- User Registration & Login
- Role-Based Authentication
- Student Dashboard
- Admin Dashboard
- Event Creation & Management
- Event Registration
- Search & Filter Events
- AI Event Recommendation
- Responsive User Interface
